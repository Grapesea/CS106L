  // step 1: Identify all iterators to space characters
  auto iter = find_all(source.begin(), source.end(), ::isspace); //[](char c){return c == ' ';}

首先，lambda函数的书写不太熟练，`[](){}`中，`[]`部分放的是调用的外部参数，`()`放的是将要被传入的参数，`{}`是函数的本体.

其次是`std::ranges::view::tranform`和`std::ranges::view::filter`的惰性求值特性我一开始完全不能理解.

(`namespace rv = std::ranges::view`这句命名空间语法可以学习一下.)

惰性求值：即只接受参数但不立即计算，在实例化的时候才开始计算.

* `std::ranges::view::transform`: 对范围中的每个元素应用一个函数，将其转换成另一种类型或值，返回一个包含转换结果的惰性视图（一个迭代器）

* `std::ranges::view::filter`: 对范围中的每个元素应用一个函数，决定它的去留，返回一个包含转换结果的惰性视图（一个迭代器）

值得积累的点：

* [`std::inserter(Container& c, typename Container::iterator i )`](https://en.cppreference.com/w/cpp/iterator/inserter.html). 其作用也着实新奇，是一个抽象函数，将传入的数据插入到`c`在迭代器`i`后面的位置.

* [`int std::isspace(int ch)`](https://en.cppreference.com/w/cpp/string/byte/isspace)：如果是空格类符号就非零返回，否则返回0.

    空格类符号：

    * space (0x20, `' '`)
    * form feed (0x0c, `'\f'`)
    * line feed (0x0a, `'\n'`)
    * carriage return (0x0d, `'\r'`)
    * horizontal tab (0x09, `'\t'`)
    * vertical tab (0x0b, `'\v'`)

    如果希望规避安全问题，可以自己将传入的`ch`用`static_cast`改成unsigned char类型.

    ```cpp
    bool my_isspace(char ch)
    {
        return std::isspace(static_cast<unsigned char>(ch));
    }
    ```

* UnaryPred: 接受一个参数、返回 `bool` 的可调用对象

    用模板可以让任何可调用对象都能传入.

    ```cpp
    // ✅ lambda、函数指针、仿函数都可以
    template <typename Iterator, typename UnaryPred> find_all(Iterator begin, Iterator end, UnaryPred pred);
    ```
